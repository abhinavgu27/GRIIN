GRIIN is an advanced infrastructure monitoring platform designed to track, analyze, and visualize road conditions in real-time. Built with a highly scalable cloud architecture, it leverages custom artificial intelligence and dynamic mapping to provide incredibly granular insights into infrastructure health.

Key Features
Custom Indian Road AI: Specialized machine learning models trained specifically to detect and assess conditions unique to Indian roads.

Real-Time Cloud Engine: A robust, highly available backend processing system that handles live data streams instantly.

Interactive Live Map: A dynamic, frontend visualization interface that plots infrastructure data and health status in real-time.

5m Granular Filter: High-precision spatial filtering that allows for incredibly detailed, 5-meter resolution analysis of road segments.

Tech Stack
Backend: Spring Boot

Database: PostgreSQL

Cloud Infrastructure: AWS / Google Cloud Platform (GCP)

AI/ML: Custom Road Assessment Models
